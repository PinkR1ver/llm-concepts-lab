# Project-local skills

This folder is reserved for project-specific or manually installed skills used by `llm-concepts-lab`.

Examples:
- design / UI refinement skills
- visualization-focused helper skills
- teaching-structure skills
- code-generation workflow helpers scoped to this project

Conventions:
1. Keep skills project-scoped when they encode assumptions specific to this lab.
2. Prefer documenting why a skill was added in `.agents/memory/`.
3. If a skill meaningfully changes project workflow, update `.agents/AGENTS.md`.
4. Impeccable-related skills are installed locally for this project and should be considered when refining UI, layout, typography, and design quality.
